# Benjamin

Building tools at the intersection of AI and marketing.

**SEO audit tools** for AI-powered CLIs - [gemini-seo](https://github.com/avalonreset/gemini-seo) and [codex-seo](https://github.com/avalonreset/codex-seo) bring professional search optimization to your terminal.

**AI video** - [wan2gp-operator](https://github.com/avalonreset/wan2gp-operator) runs Wan2GP text-to-video with VRAM-aware batching and auto-retry.

**Terminal** - [BenjaminTerm](https://github.com/avalonreset/BenjaminTerm) is a hacker-styled WezTerm distribution for Windows with smart clipboard and 86 dark themes.
